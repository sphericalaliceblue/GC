# 【表面】AIって、なぜ気をつける必要があるの？

## AIに情報を入力するということは…

AIに何かを入力することは、**見知らぬ誰かに手紙を渡すようなこと**です。

一度渡した手紙は、取り返せません。

---

## こんなことが起きる可能性があります

| リスク | 身近な例で言うと… |
|--------|-------------|
| 情報漏洩 | お客様の名前や現場の情報が、知らない人の手に渡ってしまう |
| 契約違反 | 「秘密にします」と約束していた情報を漏らしてしまう |
| 著作権違反 | AIが作った文章や画像が、誰かの作品を無断でまねたものである場合がある。気づかず使うと問題になる |
| 信頼の喪失 | 取引先・お客様からの信頼を失い、仕事を失う |
| AI出力の誤り | AIは自信たっぷりに間違えることがある。確認なしに使うと事故になる |

---

## だから、入力前に3秒考えてください

```
┌────────────────────────────────────────────┐
│                                            │
│  🟢 入れていいもの                          │
│     → 会社のホームページに載っている情報    │
│     → 世の中に公開されている情報            │
│     （インターネットで誰でも見られる情報）   │
│                                            │
│  🟡 上司に確認してから                      │
│     → 社内の資料・手順書・連絡メール        │
│     （社内の人だけが知っている情報）        │
│                                            │
│  🔴 絶対にダメ（送ったら取り消せない！）    │
│     → お客様の名前・住所・電話番号          │
│     → 工事現場・客先の写真・図面           │
│     → 見積金額・契約内容                   │
│     → 社員の個人情報                       │
│     → パスワード・IDなどの認証情報         │
│                                            │
└────────────────────────────────────────────┘
```

🔴が「絶対ダメ」な理由——それは、**漏れたら取り返しがつかない情報**だからです。

---

### 迷ったときはこの一言

> **「これが外に漏れたら、お客様や会社が困る？」**
> → YES なら 🔴　絶対に入力しない
> → NO  なら 🟢 か 🟡（上司に確認）

# 【裏面】何をしていい？何がダメ？

## やっていいこと・ダメなこと

| ✅ やっていいこと | ❌ やってはいけないこと |
|----------------|-------------------|
| 公開されているルールや法令を調べる | 客先・現場の写真をAIに送る |
| 公開中のカタログの文章を整える | 取引先名・金額が入った書類を貼り付ける |
| メールの文章を整える（名前や会社名を消してから） | AI出力をチェックせずそのまま提出する |
| 公開情報を使って勉強・調査する | 個人アカウントで業務利用する |
| 議事録の文章を読みやすく整える（社外秘でないもの） | 現場での会話・打合せをそのまま録音してAIに送る |

## 使っていいAIツール

| 使っていいツール | アカウント |
|--------------|----------|
| X-bot（Dify） | 会社アカウント |
| ChatGPT | 会社アカウント |
| notebookLM | 会社用メールアドレス ※公開情報のみ |
| Gamma | 会社アカウント |
| イルシル | 会社アカウント |
| それ以外 | → DX推進事業部に申請してから |

**⚠️ 個人のアカウントで業務に使うのはNGです**

---

## AI利用ルール　5つのポイント

```
① 使うツールは決められたものだけ（会社アカウントで）
② 入力前に「漏れたら困る情報か？」と3秒考える
③ AIの出力は必ず自分で読んで確認する
④ 迷ったら上司かDX推進事業部に確認する
⑤ ミスしたらすぐ報告する（隠さない）
```

---

## 困ったときは一人で悩まず連絡を

```
▶ まず：上司・現場責任者に確認

▶ それでもわからない場合：
　 DX推進事業部
　 TEL：（後で記載）
　 メール：（後で記載）
```

**ミスしてしまったときも、すぐに報告してください。**
隠さないことが一番大事です。早めに報告するほど、被害を小さくできます。
